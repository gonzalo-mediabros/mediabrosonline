FaqAccordion.astro

Acordeón con details nativo

Usa details/summary HTML nativo. Open state con selector CSS open:. Sin JavaScript. El ícono + / − se maneja con group-open:.

Ver código arrow_forward

¿Qué estilos tiene el summary en estado cerrado?

add remove

text-md lg:text-lg font-display font-bold text-brand-foreground. flex items-center justify-between. cursor-pointer. list-none para ocultar la flecha nativa del browser. py-4 lg:py-6 de padding vertical.

¿Cómo cambia el fondo al abrir?

add remove

El details tiene open:bg-brand-primary/5 open:pl-8 open:hover:bg-brand-primary/5. El hover en estado cerrado es hover:bg-brand-dark/2. Transición con transition-all.

¿Dónde va el contenido de la respuesta?

add remove

En el slot del componente. El div contenedor tiene mt-0 mb-8 text-sm leading-normal md:text-md text-brand-muted. Puede recibir HTML arbitrario vía slot, incluyendo párrafos, listas o código.

¿El separador inferior siempre está visible?

add remove

Sí — border-b border-brand-gris/10 en el details. Si querés quitar el último separador, hay que manejarlo desde el padre con last:border-b-0 o un wrapper.

¿Cómo se usa en la página?

add remove

Se importa FaqAccordion y se usa como <FaqAccordion question="...">respuesta</FaqAccordion>. Puede ir dentro de un layout 2-col como este (sticky left + scroll right) o en columna simple con max-w acotado.

CardSection.astro — wrapper component

Componente de tarjeta versátil

variant="light" glowStyle="none"

Tarjeta Light

Sin glows, fondo claro. Perfecto para secciones de estadísticas o información.

variant="primary" glowStyle="single" + pattern

Tarjeta Primary - Single Glow

Con patrón ondulado y un glow en la esquina superior derecha. Ideal para secciones destacadas.

variant="primary" glowStyle="dual" + pattern

Tarjeta Primary - Dual Glows

Con dos glows (superior derecho e inferior izquierdo) creando efecto envolvente. Uso en hero sections.

variant="dark" glowStyle="dual"

Tarjeta Dark - Dual Glows

Fondo oscuro profundo con glows de marca. Perfecta para CTAs y secciones enfocadas.
